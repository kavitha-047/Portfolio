## üöÄ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn
- A Supabase account

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/kavitha-047/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Set up Environment Variables**:
   Create a `.env` file in the root directory and add your Supabase credentials:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

4. **Run the development server**:
   ```bash
   npm run dev
   ```

## üì¨ Contact Form Setup

This project uses a `messages` table in Supabase. Ensure your table has the following structure:

| Column | Type | Default |
| :--- | :--- | :--- |
| id | bigint | Generated by default |
| created_at | timestamptz | now() |
| name | text | - |
| email | text | - |
| message | text | - |
